# PARSER_PHP

- Эта задача состоит из двух подзадач

## Работа с excel файлами при помощи библиотеке `PhpSpreadsheet-1.12.0`

### Базовый возможности

1. Создание excel файла.
1. Добавление данных в ичейки.
1. Редактирование стиля ячейки.
1. Скачивание созданого файла.

#### Создание excel файла

Чтобы создать файл нужно:

1. Подключить библиотеку
1. Создать экземпляр класса (виртуальный excel).
1. Обратится к активной странице excel.

```php
<?php
// подключение библиотеки
requore_once "./путь_до_autoload.php";

// сокрашения 
use \PhpOffice\PhpSpreadsheet\Spreadsheet;

// создание экземпляра класса
$spreadSheet = new Spreadsheet();

// обрашаемся к активной странице
$sheet = $spreadSheet->getActiveSheet();
;
```

#### Добавление данных в ячейки

Чтобы добавить данные нужно выполнить коснтрукцию типа:

```php
$sheet -> setCell('адрес', 'данные');
$sheet -> setCell('A1', 'Товар');
```
